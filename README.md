# Cyber Threat Detection System

# Project Description
This project is an AI-inspired Cybersecurity Analytics System that detects suspicious activities such as failed logins, malware attacks, and IP anomalies.

# Features
- Log analysis
- Malware detection
- Suspicious IP tracking
- User activity analysis
- Attack visualization
- Alert generation

# How to Run
1. Install requirements:
   pip install pandas matplotlib seaborn plotly scikit-learn streamlit

2. Run main file:
   python main.py

# Project Structure
- analyzer.py → Data analysis
- visualization.py → Graphs
- predictor.py → Alerts
- main.py → Main execution
